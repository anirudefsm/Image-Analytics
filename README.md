# WreckCheck AI: Vehicle Damage Analyzer 🚗 

## 📌 Overview
This project is an AI-powered system that detects car damages from images and estimates repair costs using YOLOv8 object detection.

## 🚀 Features
- Detects multiple types of car damage
- Provides bounding box visualization
- Estimates repair cost
- Interactive UI using Gradio
- Deployed on Hugging Face

## 🌐 Live Demo
👉 https://huggingface.co/spaces/RR38/car-damage-detection

## 🧑‍💻 How to Use
1. Open the live app link  
2. Upload a car image  
3. Click "Analyze Damage"  
4. View results and cost estimation  

## 🧠 Model
- YOLOv8 (Ultralytics)
- mAP: ~0.71
- Trained on 22 classes

## 🛠️ Tech Stack
- Python
- YOLOv8
- OpenCV
- Gradio
- Hugging Face

## 📊 Model Performance
- mAP@50: ~0.71  
- Precision: ~0.79  
- Recall: ~0.65  

## 💼 Business Use Cases
- Insurance claim automation  
- Vehicle inspection  
- Car rental damage verification  

## ⚠️ Limitations
- Lower accuracy for rare classes  
- Sensitive to image quality
